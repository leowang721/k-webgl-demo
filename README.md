# k-webgl demo

a simple demo using k-webgl.

## Install

```
npm install
```

## Run Local Server

```
sh tool/local.sh
```

## Contains

- Point
- Line
- Triangle
- Mixed Shapes
- Transform
- 3D
- Objects, such as a simple Miku Model
